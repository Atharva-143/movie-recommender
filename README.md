# 🎬 Movie Recommender System

A Movie Recommendation System built using Machine Learning and Flask. This project suggests similar movies based on content similarity and displays movie posters, ratings, and genres.

---

## 🚀 Features

- 🔍 Search movies with auto-suggestions
- 🎯 Get top 5 similar movie recommendations
- 🖼️ Displays movie posters
- ⭐ Shows ratings
- 🎭 Shows genres
- 🔗 Click on movie to view details

---

## 🛠️ Technologies Used

- Python
- Flask
- Pandas
- Scikit-learn
- HTML, CSS, JavaScript
- OMDb API
